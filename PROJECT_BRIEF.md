# 루트제이 — 제품 브리프

앱 이름: **루트제이** (Route J · J가 짜는 루트 / 루트를 제의)

## 권한

| 상태 | 열람 | 좋아요·찜 | 루트 작성 |
|------|------|-----------|-----------|
| 비회원 | 인기 샘플 (홈 10 + 카테고리당 3) | ✗ | ✗ |
| 회원 | 공개 전체 | ✓ | ✗ |
| 커플 | 공개 + 우리 탭 | ✓ | ✓ (양쪽 수정) |

## 루트

- 제목, 소개, 순서 장소, 지역·역·테마(프리셋)
- 장소당 사진·별점 선택
- 공개 / 우리만 보기 / 삭제
- 인기: 좋아요×5 + 조회×1

## 커플

- 6자리 코드, 1:1 고정
- 링크 초대는 2차

## 기술

- Expo, Supabase, 카카오(2차)
