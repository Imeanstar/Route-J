# 성수권 마케팅 (Phase 1)

## 집중 권역

- **1차**: 서울 성수·건대 (`partner_venues.region_name` 기준)
- **2차**: 홍대·연남, 한남 (검색 탭 인기 카드와 동일 메시지)

## 채널

1. 지인 20~50커플 — TestFlight / Play 내부 테스트 (`eas.json` preview 프로필)
2. 인스타·스레드 — 「우리 데이트 코스」스크린 + `shareRoute` 딥링크
3. 오프라인 — 파일럿 제휴 매장 QR 스티커 (쿠폰 `ROUTEJ10`)

## 온보딩 메시지

앱 첫 실행 → 인기 코스 탐색 → 커플 연결 → 첫 루트 작성 (`OnboardingModal`)

## 측정 (스프레드시트)

- 가입 → 커플 연결 %
- 커플당 월 루트 수
- D7 / D30 (앱 오픈 이벤트 `app_open`)
- 공개 루트 조회·좋아요 (`view_count`, `like_count`)

## 성공 기준 (조정 가능)

MAU 커플 500+, 공개 루트 200+, 주간 신규 루트 30+
